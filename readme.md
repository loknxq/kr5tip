# Таймер обратного отсчёта

## Принцип работы

- Приложение поднимает Express-сервер и раздаёт статические файлы из папки `public` (HTML, CSS, JS).
- На странице пользователь вводит длительность таймера в секундах и отправляет форму.
- Фронтенд через `fetch` отправляет `POST /api/timer` с длительностью, сервер создаёт таймер и возвращает его id.
- Скрипт на странице раз в секунду запрашивает `GET /api/timer/:id`, обновляя оставшееся время.
- Когда время заканчивается, показывается сообщение об окончании, блок таймера скрывается, а на сервере его можно удалить через `DELETE /api/timer/:id`.

1. Клонируйте репозиторий в любую папку:
3. Установите зависимости:

   npm install

4. Запустите приложение:

   http://localhost:3000

После этого отобразится страница с формой для ввода длительности таймера.
